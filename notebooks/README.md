# Notebooks

Use this folder for exploratory analysis and model comparison artifacts used in demo.

Recommended notebooks:
- `01_data_profile.ipynb` - class distribution, sample rows, source split.
- `02_baseline_vs_embedding.ipynb` - TF-IDF baseline vs MiniLM metrics.
- `03_confusion_analysis.ipynb` - confusion matrix + error analysis.

If you prefer scripts over notebooks, keep generated charts/exports here and reference them in the slide deck.
